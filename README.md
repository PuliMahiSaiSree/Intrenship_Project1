# 📋 Simple Task Manager

A clean and responsive Task Manager web application built using **HTML, CSS, and JavaScript**. This application allows users to add, complete, delete, and manage daily tasks through an intuitive user interface.

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete individual tasks
- 🗑️ Clear all tasks at once
- 📊 Real-time task statistics
  - Total tasks
  - Completed tasks
  - Pending tasks
- 📱 Fully responsive design
- 🎨 Modern glassmorphism-inspired UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📸 Project Preview

Task Manager interface includes:

- Task input field
- Add Task button
- Task counter
- Completed/Pending statistics
- Task list with checkbox support
- Delete and Clear All functionality

## ⚡ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

No additional installation or dependencies are required.
